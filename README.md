# MusicByte

**Your essential offline companion for mastering music theory and audio engineering.**

A sleek, dark-themed toolkit built for musicians, producers, guitarists, and audio professionals.

![MusicByte Logo](assets/images/MusicByteLogo.jpg)

## ✨ Key Features

- **Interactive Circle of Fifths** — Swipe to explore keys and harmonic relationships
- **Chord & Scale Viewer** — Diatonic chords and scales with guitar voicings + piano keyboard visualization
- **Guitar Chord Diagrams** — Inversion-aware, playable shapes
- **Audio Engineering Reference** — Professional instrument guides (EQ, compression, tracking, mics, mixing tips) for guitar, bass, drums, vocals, piano, and more
- **Practice Log** — Calendar view, weekly stats, streaks, timed sessions with notes
- **Powerful Tools**:
  - Metronome (subdivisions + accent downbeats)
  - Chromatic Tuner (microphone input)
  - Ear Trainer
  - EQ Cheat Sheet & Frequency Reference
  - Pro Tools Shortcuts
  - Tone Generator

**Fully offline • No login required • Beautiful dark UI**
